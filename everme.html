Project Overview

The Story

EverMe is a website where memorial pages can be created, regardless of whether it is about a deceased relative or the beloved pet.

In addition to the possibility to create memorial pages, there is also a store where a created memorial page can be connected with a QR code product for the gravestone. The website is thus intended to help the mourner in a convenient way to commemorate in a timely manner and to preserve memories.

Bei EverMe handelt es sich um eine von mir erstellte fiktive Webseite. Ich habe die Webseite als freiwilliges Übungsprojekt 2021 in wix.com gestartet. Nach dem UX Design Kurs über die Plattform CareerFoundry habe ich das Projekt nochmal neu in Figma Redesigned. In diesem Prozess habe ich nachträglich User Research betrieben und User Personas erstellt. Bei der Erstellung der Designs habe ich mich zudem hauptsächlich an Richtlinien von Material Design orientiert.

User Research

Competitive analysis

To begin, I researched for similar grief sites that are currently on the market and analyzed them in their structure and features. Since the market is saturated and many websites are similar in their basic principle, I focused my analysis on certain factors like:

  • Popularity
  • Special/unique features
  • Reviews

Analyzed websites

  • memoria.com
  • e-memoria.com

Gedenkseiten.de is by far the most popular and well-known website for memorial pages. It has by far a high number of active users compared to similar services.

e-memoria.de is a lesser-known website, but is characterized by its special function of providing QR codes for the gravestone with the memorial pages created. A function that has a particularly positive effect on the ratings.

User Stories

  • As a user, I would like to be able to add other media such as pictures, videos and poems in addition to memorial texts to make the memorial page more personal and to capture special moments and memories of the deceased.
  • As a loved one, I would like a digital QR code to be on the headstone so that the life of the deceased can be shared with those who wish it not to be forgotten.
  • As an elderly user, I would like to be able to easily navigate a clear menu on the website to be able to create a memorial page without much effort or technical knowledge.
  • As a user I want to be able to choose whether the animal memorial pages or the human memorial pages are displayed, to see only the content that interests me and not get lost in the confusion.

Design

Redesign

Originally I created the website using the platform wix.com in a modular system. For training purposes I decided to redesign the website in Figma. I used the known guidelines of Material Design for the design.

The Structure

During the redesign process I kept most of the structure of the website. With the information from the analyses, user stories and user personas, I was able to add useful details and information to the website in order to create a product that always has the user and his wishes in mind.

To simplify the process, I created most of the elements and components using auto layouts and constraints. This makes it much easier to add or delete elements after the fact, as the entire design adjusts automatically.

Conclusion

User centered design

Although I have already started this project once before via the wix.com platform, I have been able to develop the project enormously in its function and presentation. Even though both versions are still similar in their basic structure, some usability problems of the predecessor could be uncovered and corrected.

Findings

Starting the project again was a lot of fun and showed me that there is always room for improvement. Even now the project is still expandable. Many of the functions I have not developed to the end, because it would have blown up the time frame I had available.

Also, this project makes it clear to me once again that UI and UX design should not be considered separately, but belong together. In interaction, problems can be identified and the maximum potential of an application can be extracted.
